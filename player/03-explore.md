# 探索秘境（卷轴 + 精力）

## 单次探索
- **初级**：**初级卷轴 + 2×初级精力（6）** → **产值≈12.47** → **理想利润≈2.24/次**。  
- **中级**：**中级卷轴 + 2×中级精力（24）** → **产值≈36.86** → **理想利润≈3.68/次**。

## 日卷轴产出（以 12 个一级宝箱为例）
- 初级卷轴 **≈10.85** / 日；中级卷轴 **≈0.71** / 日。  
- 若直接出售卷轴，**约 26.96/日**（理想成交）。
